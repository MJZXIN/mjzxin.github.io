Hexo 安装教程 1

系统准备

  • Git
  • Node.js

Git 可以从官网下载,更新时(2020.08.31)是2.28.0版.

Node.js也可以从官网下载,文章更新时(2020.07.27)是12.18.3 LTS版.

安装Hexo

打开 Git bash

  1. 定位到我们新建的项目文件夹,如 D:\Hexo.
  1. 安装Hexo
  1. 初始化Hexo,将会拉取Hexo的基础文件 (以后如果操作失败,可以直接删除文件夹内的所有文件,从头再来)
  • 如果网络慢,可以使用淘宝镜像代替. 可以进行验证.

​ 如果返回 https://registry.npm.taobao.org，说明镜像配置成功。以后可以用cnpm代替npm.

  1. 可以查看已安装的版本信息.
  1. 运行已安装的Hexo,按顺序运行.

试运行

打开浏览器,输入 localhost:4000进入,就可以看到 hexo 的面容了.
